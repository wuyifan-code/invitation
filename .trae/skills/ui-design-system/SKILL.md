# UI Design System Generator

## Overview

This skill creates comprehensive UI design systems for web and mobile applications. It generates design tokens, component libraries, and design guidelines that ensure consistency across your project.

## Features

- **Design Tokens**: Generate consistent color palettes, typography scales, spacing units, and other design variables
- **Component Library**: Create reusable UI components with consistent styling
- **Design Guidelines**: Provide best practices and usage instructions
- **Responsive Design**: Ensure components work across different screen sizes
- **Accessibility**: Include accessibility considerations and guidelines

## Usage

### Basic Usage

When creating a new project or redesigning an existing one, invoke this skill to generate a complete UI design system. The system will include:

1. **Color Palette**: Primary, secondary, accent, and neutral colors with hex, RGB, and HSL values
2. **Typography System**: Font families, sizes, weights, and line heights
3. **Spacing System**: Consistent spacing units for layout and components
4. **Component Library**: Buttons, forms, cards, navigation, and other common UI elements
5. **Design Guidelines**: Usage instructions and best practices

### Example Prompt

```
Create a UI design system for a modern e-commerce website with a clean, minimalist aesthetic. Include color palette, typography, spacing, and common components like buttons, product cards, and navigation.
```

## Output Format

The design system will be delivered as:

1. **Design Token Files**: CSS variables, SCSS variables, or design token JSON files
2. **Component Examples**: HTML/CSS examples of each component
3. **Design Guidelines**: Markdown documentation with usage instructions
4. **Style Guide**: A visual reference of all components and design elements

## Best Practices

- Start with a clear brand direction and target audience
- Define core design principles before generating the system
- Test components across different devices and screen sizes
- Iterate based on user feedback and usability testing

## Customization

The design system can be customized based on:

- Brand identity and color preferences
- Target platform (web, mobile, or both)
- Industry-specific requirements
- Accessibility needs
- Performance considerations

## Integration

The generated design system can be integrated with:

- Frontend frameworks (React, Vue, Angular, etc.)
- CSS preprocessors (Sass, Less, PostCSS)
- Design tools (Figma, Sketch, Adobe XD)
- Component libraries (Bootstrap, Material UI, etc.)
